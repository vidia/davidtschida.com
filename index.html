Purdue Laundry

ACM Sigapp has released a Laundry app for Purdue students! It allows you to check availabilities of laundry machines in Purdue dorms and set timers so you never forget your laundry. It is developed by members of Sigapp and is completely open source on GitHub.

Purdue App

Also completed as a part of ACM Sigapp, this app was once the official app of Purdue University. It is slated for a rerelease after reviews of the replacement app started to fall. It provides links and access to Purdue resources that are important to students, from busses to dining courts.

Hubless IoT

An IoT suite, containing code for and Android client and microcontroller, that provides cheap, hubless, access to the IoT space. It is backed by AWS IoT, but adds a cleaner API and a frictionless setup experience for potential customers.

Dining Courts

An android application to access Purdue Dining court menus in a more intuitive way. Right now every application's easiest interation is to check various DAY's menus, when checking various LOCAITON's menus is much more common. This app aims to make checking foods at various locations more seamless. It is clated for release to the Play store soon.
